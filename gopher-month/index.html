Gopher Month

Go, also called golang, is an open source programming language (though initially developed at Google) that makes it easy to build simple, reliable, and efficient software.

A Gopher stands for the Go project and Go programmers everywhere.

Go has:

  • Simple, minimal syntax
  • Garbage collection built-in
  • A flexible interface system
  • Easy concurrency support via goroutines
  • Fast compilation times
  • Simple compile build/run procedures
  • Statically linked binaries that are simple to deploy
  • Fun to write
  • Much much more..

GopherConIndia 2015 will be India's first-ever Go programming conference. The conference is expected to attract the best Go programmers across the country and abroad.

GopherConIndia is celebrating the month of January - 2015 as Gopher Month. The aim is to teach and increase the awareness of Go Programming Language within the student community in India.

HOW YOU CAN HELP

Gopher month will be organized and run by volunteers. Anyone wanting to participate is welcome and all help is appreciated.

  3. Help us in coordinating Gopher Month by talking to colleges to arrange Go workshops.
