WHAT IS GOPHERCON INDIA ?

The Go Conference in India

This conference is brought to you by the Go Language Community in India together with the Emerging Technology Trust (ETT). ETT is a non-profit organization, established to organize and conduct technology conferences in India. It's current portfolio includes DeccanRubyConf, DevOpsDaysIndia, DroidJamIndia, GopherConIndia and RubyConfIndia.

We love going to conferences that don't make you choose between two great presentations. So we created GopherConIndia as a single-track event that you don't want to miss and where everyone gets the opportunity to see the same talks. We think you'll enjoy the speakers we're lining up so much, you won't want to miss anyone of them.

Note: The conference is entirely in English.

Our Speakers

From Polling to Pushing: Going Realtime with Go, gRPC, and Redis
Traditional request-response models suffer from delays and high backend load due to polling. This talk covers building and scaling Propeller at CRED, a real-time, bidirectional communication platform between frontend and backend using Go, bidi gRPC, and Redis PubSub.
Abhishek Varshney
Architect at CRED
Think Pragmatically: Elevating Your Go Development Skills
Learn how pragmatic programming enhances software development. This talk covers simplicity, effective tooling, and collaboration for building robust applications. Discover real-world examples and best practices to achieve cleaner code, better team dynamics, and more successful projects.
Ayush Goyal
Sr. Principal Software Engineer at PubMatic
Paradigms of Rust for the Go developer
The talk delves into 3 key paradigms: Go's CSP model for concurrency, Rust's data race prevention through ownership and borrowing, and Rust's opt-in shared memory model. With practical examples and comparisons, attendees will learn how these paradigms foster robust, efficient, and safe software.
Gaurav Gahlot
Senior Software Engineer at Acquia
Mastering Concurrency in Go: From Patterns to Production
Unlock the full potential of Go concurrency model with advanced patterns like pipelines, fan-out/fan-in, and graceful cancellation. This talk offers practical insights and real-world examples to help developers build scalable, efficient systems while avoiding common concurrency pitfalls.
Joobi S B
Sr. Tech Lead at KeyValue Software Systems
Billion Rows Walk Into a Go Bar
1BRC (1 Billion Row Challenge) is about parsing a file with a billion rows to find aggregations as fast as possible. In this talk, Iâ€™ll share my analysis of exploring various solutions to this problem and how you can approach writing highly performant Go programs in your day-to-day work.
Mohit Kumar
Software Engineer at One2N
Building a recommendation engine for Bus search based on Real-time feedback of users
Discover how we boosted RedBus conversions by 20% using real-time data and Gradient Boosting Regression Trees (GBRT). Learn how Redis sorted sets enabled ultra-fast, personalised bus rankings. Our model ranks buses using user interactions, enhancing both experience and occupancy rates.
Pawan Kumar Sharma
Associate Principal Engineer at Redbus
Empowering Resilient Scalability: The Power of Choreography in Distributed Systems
Choreography in distributed systems: Decentralized interactions, resilient scalability and graceful failure handling. Druva's innovative event framework showcases these principles, facilitating seamless communication and robust application performance. Join us to explore the future of system architecture!
Rohit Ekbote
Lead Engineer at Druva
Questioning database claims: Design patterns of storage engine
How storage engine design impacts database durability, performance, and consistency. This talk provides practical insights to critically evaluate database claims and ensure they meet your needs.
Sarthak Makhija
Principal architect, Caizin
The Loop War: Iterators vs. Traditional Loop | A Generics
Unlock the future of data traversal with Go 1.23's new iterators! In my talk, I'll reveal how to harness the power of the elegant for range over functions to iterate over custom data structures seamlessly. Elevate your Go code with cleaner patterns and efficient algorithms. Don't miss out!
Sanjay Shiradwade
IBM
Sidharth Sasikumar
Sr. Software Engineer at Synamedia
Operationalising golangci-lint
Talk shows how to leverage golangci-lint in Go projects for catching issues early, ensuring code consistency, and preventing bugs. We'll explore why we should configure golangci-lint and how to integrate it into existing codebases, equipping you to deliver cleaner, more maintainable code.
Suhas Karanth
Staff Engineer at Netskope
Workflow library in Go for API development
This talk explores how the built-in Go workflow library simplifies API development, optimizes performance, and tackles challenges such as unstructured logic and code duplication. We'll delve into its architecture, use of goroutines and channels, and demonstrate how it streamlines complex APIs.
Suryanshu Singh
Lead Software Engineer at MakeMyTrip
Contextual Logging for Modern Go Applications
Contextual logging lets developers add meaningful context to log entries, making them more informative for debugging distributed systems. This talk will show how Go developers can use contextual logging to streamline logs, cut noise, and speed up troubleshooting.
Tapan Avasthi
Cyberhaven Inc.

Getting to Jaipur

By Air:

Jaipur International Airport (JAI) is approximately 4 km from the hotel, with taxis and ride-sharing options like Uber and Ola readily available. Travel time is around 15-20 minutes.

By Train:

Jaipur Junction Railway Station, about 10 km from the hotel, connects Jaipur with major cities. Taxis and auto-rickshaws are available, with a 30-minute drive to the hotel.

By Road:

Jaipur is accessible via NH8 from Delhi (about 5 hours) and NH21 from Agra (about 4 hours).

Passports and Visas

International Visitors: Most international attendees require a visa to enter India. Visitors from several countries may be eligible for an e-visa. For details and to apply online, please visit the Indian e-Visa Portal.

Documents Required: Ensure you carry a valid passport (with at least six months validity) and other identification documents while traveling.

The City of Jaipur

Jaipur, also known as the "Pink City," is the capital of Rajasthan. Famous for its vibrant culture and stunning architecture, Jaipur combines ancient charm with modern amenities. Popular landmarks include:

Amber Fort
City Palace
Hawa Mahal
Jantar Mantar

Jaipur is also known for its handicrafts, jewelry, and textiles, making it an excellent place for shopping.

Safety Tips
  • Stay alert and secure your belongings in crowded areas.
  • Use registered taxis, hotel shuttles, or ride-sharing apps like Uber or Ola for transportation.
  • Save emergency contact numbers (112) and embassy details.
